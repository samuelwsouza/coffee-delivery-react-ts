# React + TypeScript + Vite

```
<h2>Coffee Delivery</h2>
<h3>Coffee Delivery é uma plataforma de pedidos de café online, onde os clientes podem fazer pedidos de café com facilidade e conveniência, e receber suas bebidas favoritas diretamente em suas portas.
</h3>

<h4>Tecnologias Utilizadas</h4>

 - React: Uma biblioteca JavaScript para construção de interfaces de usuário.
 - React Router DOM: Uma coleção de componentes de navegação declarativos para React.
 - Styled Components: Uma biblioteca para estilizar componentes React com CSS-in-JS.
 - React Hook Form: Uma biblioteca para gerenciar formulários em React com ganchos.
 - @hookform/resolvers: Resolvedores para React Hook Form, utilizados para validação de formulários.
 - Zod: Uma biblioteca de validação de esquemas para TypeScript e JavaScript.
 - Immer: Uma biblioteca que permite trabalhar com cópias imutáveis do estado de forma mais fácil.
 - Phosphor React: Uma biblioteca de ícones SVG para React.
 - Eslint: Uma ferramenta de linting para identificar e reportar padrões problemáticos no código JavaScript.

git clone https://github.com/samuelwsouza/coffee-delivery.git

npm install
Inicie o servidor de desenvolvimento:

npm start
Abra o navegador e acesse http://localhost:3000 para visualizar o aplicativo.

Contribuições são bem-vindas! Sinta-se à vontade para abrir um problema ou enviar uma solicitação de pull.
```
